# Portugal-Road-Accidents-Report

This report reflects the development of road accidents in Portugal from 2014 to 2023, taking into account the regions where the accidents occurred, the type of road, and the number of injuries and deaths.

# KPI's

  - Total Current Year Accidents
  - Total Current Year Deaths
  - Total Current Year Injureds

# Graphs

  - Death Casualties by Year
  - Accidents Growth by Year
  - Total Accidents By Region
  - Total Death Casualties by Region
  - Death Casulaties by Region & Roads
